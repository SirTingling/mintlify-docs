Configure your brand to appear throughout the client experience.

## Logo setup

Upload two versions of your logo:

| Version | Usage |
|---------|-------|
| Light mode | Used on white/light backgrounds |
| Dark mode | Used on dark backgrounds |

### Logo requirements

- **Format:** SVG preferred, PNG accepted
- **Size:** Minimum 200px width
- **Background:** Transparent
- **Aspect ratio:** Horizontal logos work best

### Uploading logos

1. Go to **Firm Settings → White Label → Branding**
2. Click **Upload Logo** for each version
3. Preview how it looks in the portal
4. Save changes

<Tip>
If you only have one logo version, upload the same file for both. We'll adjust contrast automatically.
</Tip>

## Favicon

The favicon appears in browser tabs:

- **Format:** ICO, PNG, or SVG
- **Size:** 32x32 pixels (we'll resize larger files)
- **Background:** Transparent recommended

## Brand colors

Set your primary and secondary brand colors:

### Primary color

Used for:
- Buttons
- Links
- Active states
- Highlights

Enter as hex code (e.g., #2563EB).

### Secondary color

Used for:
- Secondary buttons
- Hover states
- Accents

### Background colors

Optionally override:
- Page background
- Card background
- Sidebar background

<Warning>
Ensure sufficient contrast between text and backgrounds for accessibility.
</Warning>

## Typography

Choose fonts for headings and body text:

### Available fonts

- System fonts (fastest loading)
- Inter (Pluvel default)
- Plus Jakarta Sans
- Work Sans
- DM Sans
- Custom (provide URL to self-hosted font)

### Font settings

- Heading font
- Body font
- Font weight preferences

## Preview

Before publishing, preview your branding:

1. Click **Preview Portal**
2. See how clients will experience your brand
3. Test both light and dark modes
4. Check on mobile viewport

## Publishing changes

After configuring:

1. Review all settings
2. Click **Publish Changes**
3. Changes go live immediately

Allow a few minutes for caches to clear. Some clients may need to refresh.

## Per-client branding

Want different branding for different clients? This is available on Firm Scale:

1. Go to the client's settings
2. Override specific brand elements
3. That client sees custom branding

Useful if you operate multiple brands or have enterprise clients wanting their branding.

## Email branding

Customize email appearance:

| Element | Customization |
|---------|---------------|
| Header logo | Your logo |
| Header color | Your brand color |
| Footer text | Your firm name and address |
| Reply-to address | Your email address |

Go to **Firm Settings → White Label → Emails** to configure.

## Brand guidelines

Once configured, generate a brand guidelines document:

1. Go to **White Label → Brand Guidelines**
2. Click **Generate PDF**
3. Download a summary of your brand settings

Share with team members to ensure consistent application.
