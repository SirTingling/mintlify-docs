## Customizing the view

### Show/hide columns

Not everyone needs to see AR and AP. Customize:

1. Click the gear icon in the grid header
2. Check/uncheck columns
3. Drag to reorder
4. Columns save for your account

### Saved views

Create views for different workflows:

1. Set up your filters and columns
2. Click **Save View**
3. Name it: "Month-end review" or "High-touch clients"
4. Access saved views from the dropdown

Examples:
- **Month-end review** â€” Status: needs attention, sorted by last activity
- **Priority clients** â€” Filtered to your top-tier clients
- **My clients** â€” Filtered to clients assigned to you

## Making the overview work for you

**Check it every morning.** 30 seconds scanning the grid tells you what needs attention today.

**Sort by status + last activity.** Clients in trouble who haven't been touched = highest priority.

**Use filters aggressively.** You don't need to see green clients when you're putting out fires.

**Watch the aggregate metrics.** If "Pending items" keeps growing, you're losing ground. If it's shrinking, you're catching up.

**Set up saved views.** Don't re-configure filters every time. Save them and switch with one click.
