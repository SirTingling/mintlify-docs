We ship improvements constantly. Here's what's changed recently.

---

## February 2026

### February 10, 2026

**Banking**
- Added support for 500+ additional credit unions through Plaid
- Fixed an issue where some Mercury transactions weren't syncing

**Invoicing**
- New "Duplicate invoice" button for quickly creating similar invoices
- Invoice PDF exports now include your company logo

**Payroll**
- Added support for Hawaii state taxes
- Fixed calculation issue with California SDI for high earners

---

### February 3, 2026

**Accounting**
- New "Bank Rules" feature — create rules to auto-categorize recurring transactions
- Chart of accounts now supports custom account numbering

**Compliance**
- BOI report filing directly from Pluvel (previously linked to external site)
- Added reminders for upcoming annual report deadlines

---

## January 2026

### January 27, 2026

**Integrations**
- New Shopify integration — sync orders, refunds, and payouts automatically
- QuickBooks import now includes customer and vendor balances

**Reports**
- Cash flow statement now available
- Added date range presets (This month, Last quarter, Year to date, etc.)

---

### January 20, 2026

**Payroll**
- W-2s now available for download (for 2025 tax year)
- Added support for 1099-NEC generation for contractors
- New "Preview payroll" step before submitting

**Digital Mailroom**
- Improved OCR accuracy for scanned documents
- Added "Forward to email" action for mail items

---

### January 13, 2026

**Formation**
- Now supporting nonprofit formation in all 50 states
- Reduced processing time for Delaware expedited filings to same-day

**Billing**
- New payment method: ACH direct debit (in addition to card)
- Invoice history now shows payment method used

---

### January 6, 2026

**General**
- New onboarding flow for first-time users
- Improved performance on dashboard load (40% faster)

**Mobile**
- Transaction categorization now works on mobile web
- Fixed layout issues on iPad
